# CHANGELOG: pic-parameters

## 0.2.0 (2026-07-06)

- Add optional `holidayExclusions` array (dates) to `calendar` parameters.
- Update validator support to validate v0.2.0 parameter formats.

## 0.1.0 (2026-07-06)

- Initial release of Policy Interchange Contracts (PIC) parameter schema.
- Added temporal period constraints (from/to dates).
- Standardized value representation including scalars, decimals, and bracket schedules.
- Added period-consistency validation rules (non-overlap, contiguous, open-ended final period).
